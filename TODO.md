# Mobile Responsiveness Task

## Overview
Add mobile-friendly styles to the website by implementing media queries for screens smaller than 768px. This includes adjusting headers, hero sections, slideshows, modals, pricing cards, and other elements.

## Steps
- [x] Update home.css with mobile media queries
- [x] Update commission.css with mobile media queries
- [x] Update tos.css with mobile media queries
- [x] Test the changes by opening the website in a browser
- [x] Verify mobile responsiveness on different screen sizes

## Details
- Target screens < 768px
- Adjust headers to stack vertically and center
- Reduce margins and font sizes for mobile
- Make slideshow and modal smaller
- Adjust grid layouts to single column where needed
- Ensure buttons and links are touch-friendly
